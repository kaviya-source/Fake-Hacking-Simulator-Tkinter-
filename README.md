## About

We all like pranking our friends sometimes! 😄

This is a fun and harmless Fake Hacking Simulator built using Python and Tkinter. It creates a realistic-looking hacking animation for entertainment and learning GUI development.

⚠️ No actual hacking is performed.
## ✨ Features

- 💻 Realistic terminal animation
- ⌨️ Typing effect
- 🎲 Randomized delays
- 😂 Funny prank ending
- 🐍 Built with Python & Tkinter
- ✅ Safe and harmless

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/kaviya-source/Fake-Hacking-Simulator.git
```

Move into the project folder:

```bash
cd Fake-Hacking-Simulator
```

Run the program:

```bash
python "python prank.py"
```
